# ComplyCat

ComplyCat is an open-source Python library for regulatory compliance in the financial domain. It provides tools for automating the extraction and analysis of regulatory information from documents, helping developers streamline compliance-related processes.

## Features

- **Document Stripping:** Easily extract and structure information from financial documents.
- **AML Compliance Checker:** Implement AML (Anti-Money Laundering) compliance checks on transaction data.
